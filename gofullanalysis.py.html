GO Full Analysis



  • * FASTA file with sequences of nucleotides


  • * File that contains a list of more abstract terms intended to be associated to the more specific terms.


  • * Database for comparison purposes.


  • * We will send you an email when your process is finished.
Inputs:
FASTA file with sequences of nucleotides, a file that contains a list of more abstract terms intended to be associated to the more specific terms and a database for compar-ison purposes.
Process:
Orders the system to run all the scripts in one single process.
Output:
XML file containing the alignments of the sequences.
CSV file containing the best alignments (top hit).
CSV file containing the associations between GO terms and sequence identifiers.
CSV file containing associations between GO categories and sequence identifiers.
Tabulated file that contains the various counts per each GO term that was to be found.
PNG-format image that represents the data.
PDF file containing a report about the analysis.
