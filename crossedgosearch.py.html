Crossed GO Search



  • * CSV file containing the relation between GO categories and sequence identifiers.


  • * List of GO categories that are to be found in shared (common) sequences separated by comma e.g: GO:0043226,GO:0044456
Inputs:
CSV file containing the relation between GO categories and sequence identifi-ers and also a list of GO categories that are to be found in shared (common) sequences.
Process:
The process is a filter of all the sequences that appear in various GO categories at the same time.
Output:
A list of sequence identifiers.
